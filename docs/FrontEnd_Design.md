# Front-end Design – AI Triage System

## General UI Style
- Clean, simple medical-style interface
- Light background (white or light gray)
- Blue accents for buttons and highlights
- Clear spacing and easy-to-read layout

---

## Login Page
- Centered login card
- Fields: Username, Password
- Button: Log In
- Error label below button if login fails

---

## Dashboard
- Top bar: App title + Logout button
- Search bar (large) with search button
- Recent patients section under search bar
- Clean, simple layout

---

## Patient Search Results
- Page title: Search Results
- Table with Name, ID, Age, Last Visit
- Entire row is clickable
- Back button at top or bottom

---

## Patient Details Page
- Patient info card at the top (Name, ID, Age, Gender)
- Sections:
  1. Overview / Summary
  2. Visit History
  3. Notes
- “Generate AI Summary” button near top
- Notes section has Save + Cancel buttons
- Back to Dashboard button at bottom

---

## Mobile Optimizations (Optional)
- Navigation becomes stacked vertically
